```
输入：n = 1
输出：true
解释：20 = 1
```

示例 2：

```

```

示例 3：

```
输入：n = 3
输出：false
```

示例 4：

```
输入：n = 4
输出：true
```

示例 5：


Given an integer `n`, return `true` if it is a power of two. Otherwise, return `false`.

An integer `n` is a power of two, if there exists an integer `x` such that `n == 2x`.

 

Example 1:

```
Input: n = 1
Output: true
Explanation: 20 = 1
```

Example 2:

```
Input: n = 16
Output: true
Explanation: 24 = 16
```

Example 3:

```
Input: n = 3
Output: false
```

Constraints:

- `-231 <= n <= 231 - 1`
 

Follow up: Could you solve it without loops/recursion?
